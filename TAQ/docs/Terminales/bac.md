## Pour chercher un thème précis :

### Python

- Listes
- Dictionnaires

- Piles / Files

- Dynamique : Programmation Dynamique
- Arbres
- Graphes

### Données

- Binaire : Représentation des Données en Binaire
- Table : Données en Table
- BDD : Base De Données

### Architecture

- Systèmes : Systèmes d'Exploitation et Processus
- Réseaux : Réseaux et Routage

### Divers

- Calculabilité / Décidabilité
- Sécurisation : Sécurisation des communications
