# Annales du BACCALAURÉAT GÉNÉRAL de NSI

Les annales du bac de nsi sont disponibles sur le site [sujetdebac.fr](https://www.sujetdebac.fr/annales/specialites/spe-numerique-informatique/).

## Pour chercher un thème précis :

### Python

- **Bugs** : Gestion de bugs
- **Listes**
- **Dicos** : Dictionnaires
- **Algos** : Algorithmes
- **Tri** : Algorithmes de Tri
- **Glouton** : Algorithmes Gloutons

- **POO** : Programmation Orientée Objet
- **Piles / Files**
- **Récursivité**
- **Diviser** : Diviser pour régner
- **Dynamique** : Programmation Dynamique
- **Arbres**
- **Graphes**

### Données

- **Binaire** : Représentation des Données en Binaire
- **Table** : Données en Table
- **BDD** : Base De Données

### Architecture

- **Systèmes** : Systèmes d'Exploitation
- **Processus** : Gestion des processus
- **Réseaux**
- **Routage** : Protocoles et Algorithmes de Routage

### Divers

- **Calculabilité / Décidabilité**
- **Sécurisation** : Sécurisation des communications
